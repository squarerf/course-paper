# 进度显示模板

## 通用进度模板

每步开始时显示。用 `{step_name}` 和 `{step_status}` 替换当前步骤信息。

```
📊 课程论文生成进度
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
{各步骤状态行}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**步骤状态行格式：**
- 已完成：`✅ 第X步：{步骤名} - 已完成`
- 进行中：`🔄 第X步：{步骤名} - 进行中`
- 待执行：`⬜ 第X步：{步骤名} - 待执行`

**9个步骤：**
0. 环境检测
1. 收集需求信息
2. 文献检索
3. 文献验证
4. 论文内容生成
5. 图表生成
6. Word文档生成
7. 输出检查
8. 交付

## 子任务完成确认模板

每步完成时显示该步骤所有子任务的完成情况：

```
✅ 第X步：{步骤名} - 已完成

📋 子任务完成情况
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
{子任务类别1}
  ✅ X.1.1 {任务描述} → {结果}
  ✅ X.1.2 {任务描述} → {结果}

{子任务类别2}
  ✅ X.2.1 {任务描述} → {结果}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
所有子任务完成 ✅

是否继续下一步？
```
